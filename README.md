# Webtoon Performance Insights

This project analyzes **Webtoon platform performance data** to uncover hidden insights that can help improve user engagement, increase viewership, and support creators.  
The analysis was performed using Power BI and data downloaded from kaggle.

---

##  Key Insights

### 1. Genre Performance
- **Romance** dominates with over **20B views**, showing strong emotional connection.
- **Fantasy** and **Action** genres show potential for **cross-platform storytelling**.
- **Thriller** and **Horror** genres have low competition but loyal fans — an opportunity for niche expansion.

### 2. Weekly Engagement
- **Peak days:** Sunday → Tuesday  
- **Lowest:** Friday & Saturday  
   *Strategy:* Launch new episodes or events on **Sundays** for maximum exposure.

### 3. Most Subscribed Webtoons
| Rank | Title | Subscribers |
|------|--------|-------------|
| 1 | True Beauty | 7.49M |
| 2 | Lore Olympus | 6.60M |
| 3 | unOrdinary | 6.10M |
| 4 | Let’s Play | 4.91M |
| 5 | I Love Yoo | 4.70M |

> These titles focus on strong character development and emotional hooks.

### 4. Creator Insights
- Top authors have 5M+ subscribers across series.  
 *Opportunity:* Build **creator profiles and fan engagement tools** to leverage creator loyalty.

---

##  Strategic Recommendations

| Focus Area | Action | Expected Impact |
|-------------|---------|----------------|
| Content Scheduling | Release episodes Sun–Tue | +12–15% view increase |
| Creator Branding | Showcase top authors | +20% engagement |
| Interactive Features | Add reader polls & animations | +18% retention |
| Genre Expansion | Explore Romantic Fantasy & Horror | +10% user growth |

1. Romance drives retention; Fantasy drives discovery.

3. Mid-tier creators lack exposure — use AI-based “You Might Also Like” suggestions.  
4. Underrepresented genres like Horror or Mystery can grow fast with targeted promotion.
